O que está aqui

1) index.html  -> layout base congelado + inclusão de oraculo.js no fim do <body>

Como fazer BACKUP no GitHub (2 minutos)
---------------------------------------
A) Abre o teu repositório em github.com
B) Clica no botão verde "Code" > "Download ZIP"
C) Guarda esse ZIP — é o teu backup completo ANTES do patch.

Aplicar o PATCH (substituir ficheiros)
--------------------------------------
1) No GitHub, abre o ficheiro index.html > botão lápis (editar).
   - Apaga o conteúdo todo e cola o conteúdo do nosso index.html deste patch.
   - Desce e clica "Commit changes".
2) Abre o ficheiro oraculo.js (se não existir, cria-o no raíz com este nome).
   - Cola o conteúdo do nosso oraculo.js.
   - "Commit changes".
3) Faz refresh no telemóvel (esvazia a cache se necessário).

Notas importantes
-----------------
- Este patch não altera estilos (CSS) nem outros JS.
- O oráculo usa IDs: #oracleMsg, #btnNova, #btnCopiar, #btnWhats (já preparados no index).
- Se o iPhone continuar a mostrar páginas antigas, força limpeza:
  • Safari > Definições do Website > Remover Dados deste Website
  • Ou abre em aba privada.

Diagnóstico rápido
------------------
- No iPhone, toca "Nova mensagem" 3-4 vezes:
  • Se as frases variam e não mostram números, o oráculo está OK.
  • Se surgir sempre igual, há outro JS antigo a correr em paralelo.
    — Confere que NÃO tens <script src="oraculo.js?..."> duplicado no fim.
