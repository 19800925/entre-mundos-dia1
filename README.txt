Entre Mundos — Patch de Silêncio (seguro)
=========================================
Este patch NÃO substitui ficheiros existentes. Cria uma nova página:
  • index-silencio.html
  • styles/oraculo-silencio.css
  • js/oraculo-silencio.js

Como testar (GitHub mobile/desktop):
1) Extrai o zip.
2) No repositório, carrega estes três ficheiros mantendo as mesmas pastas.
3) Acede: https://<teu-usuario>.github.io/<repo>/index-silencio.html
4) Se ficar como desejas, podes copiar o markup e script para o teu index.html
   OU renomear index-silencio.html -> index.html (após criares backup).

Reverter:
- Como estamos a adicionar novos ficheiros, basta apagar estes 3 itens para voltar ao estado anterior.
