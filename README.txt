Como publicar no GitHub Pages:
1. Faz backup dos ficheiros atuais do teu repositório.
2. Substitui os ficheiros da app por estes.
3. Mantém a publicação no mesmo branch/folder onde já tens o site ativo.
4. Espera 1 a 3 minutos e atualiza o link no iPhone.
5. Se o iPhone mostrar versão antiga, abre em privado ou limpa cache do Safari.

O que esta versão já traz:
- Visual mais premium e cósmico
- Tabs limpas e estáveis
- Oráculo com 60 frases sem repetição até fechar o ciclo
- Respiração 4-4-6 por 5 ciclos
